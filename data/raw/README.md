# Raw data

# Contains original datasets:
- who_tb_burden.csv: WHO TB burden estimates [>1Mb] downloaded from https://www.who.int/teams/global-programme-on-tuberculosis-and-lung-health/data March 6th, 2026
- ihme_tb_prevalence.csv: IHME TB prevalence estimates downloaded from https://vizhub.healthdata.org/gbd-results/ March 6th, 2026
- ihme_tb_incidence.csv: IHME TB incidence estimates downloaded from https://vizhub.healthdata.org/gbd-results/ March 6th, 2026
- ari_estimates.Rdata: ARI estimates for Houben & Dodd (2016) shared in github repo: https://github.com/petedodd/LTBIest/blob/master/data/All_3.Rdata
